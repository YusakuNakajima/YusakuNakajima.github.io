# Profile

## 中島 優作 (Yusaku Nakajima)
**所属**: 大阪大学 / BWH Radiology Research

材料科学分野へメカトロニクスやロボティクスを応用した、ラボオートメーションの研究を行っています。
実験作業を自動化するためのロボットハンドリングや、自律計測システムの開発に取り組んでいます。
