# Home

## 中島 優作 (Yusaku Nakajima)

大阪大学 博士後期課程  
ハーバード大学医学大学院 ブリガム・アンド・ウィメンズ病院（BWH）学生研究員

**Email**: yusaku_nakajima [at] ap.eng.osaka-u.ac.jp

材料科学分野において、メカトロニクスやロボティクスを応用したラボラトリーオートメーションの研究を行っています。
従来は再現性の確保が難しかった実験プロセスに対して、高再現性かつ精密なプロセス制御を実現する実験手法や、手作業では困難な環境下での実験システムの開発に取り組んでいます。
特に、粉体の粉砕・混合およびメカノケミカル反応を対象としたロボット実験システムの構築を専門としています。
