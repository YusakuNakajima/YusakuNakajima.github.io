# 公開資料

このページでは、公開している勉強用資料をまとめています。

## [ゼロからのROS入門](https://yusakunakajima.github.io/zero_ros/)

買ってきたロボットアームを使いこなすための基礎知識から、ROS を使った高度なロボット制御（軌道制御）まで扱う実践的なスライドです。

Web 上で実際に動かしながら学べる構成にしています。
2025年8月にラボ内で開催したロボットアーム勉強会の資料をもとに、改変して公開しています。
